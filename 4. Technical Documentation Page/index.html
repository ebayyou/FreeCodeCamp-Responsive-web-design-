Bootstrap

Introduction

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains HTML, CSS and (optionally) JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Accessibility

A brief overview of Bootstrap’s features and limitations for the creation of accessible content.

Bootstrap provides an easy-to-use framework of ready-made styles, layout tools, and interactive components, allowing developers to create websites and applications that are visually appealing, functionally rich, and accessible out of the box.

Here are some examples of using Bootstrap that you can try. Let's see one by one!
  • Create a table with Bootstrap

    – Want to create a table without typing lines of code? It's easy, you just need to use the "table" class in the Bootstrap file.

  • Adjusting the appearance of images with Bootstrap

    – You can create responsive images or circular images using classes such as “.img-responsive”, “.img-rounded”, “img-circle”, and “img-thumbnail”.

  • Create a panel with Bootstrap

    – If you want to display an alert message, you can use the “alert” class that contains the warning message you want to create.

  • Creating a button with Bootstrap

    – Want to add a button? You can make it easily in Bootstrap, just use the "btn" class.

  • Create alert messages with Bootstrap

    – You can also create panels containing headers, body and footers. You do this by using the "panel" class.

React_js

Introduction

React (also known as React.js or ReactJS) is a free and open-source front-end JavaScript library[3] for building user interfaces based on UI components. It is maintained by Meta (formerly Facebook) and a community of individual developers and companies.[4][5][6] React can be used as a base in the development of single-page, mobile, or server-rendered applications with frameworks like Next.js. However, React is only concerned with state management and rendering that state to the DOM, so creating React applications usually requires the use of additional libraries for routing, as well as certain client-side functionality.

Learn React

People come to React from different backgrounds and with different learning styles. Whether you prefer a more theoretical or a practical approach, we hope you’ll find this section helpful.

  • If you prefer to learn by doing, start with our practical tutorial.
  • If you prefer to learn concepts step by step, start with our guide to main concepts.

Like any unfamiliar technology, React does have a learning curve. With practice and some patience, you will get the hang of it.

What Is React?

React is a declarative, efficient, and flexible JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called “components”.

React has a few different kinds of components, but we’ll start with React.Component subclasses:

return (

}
}
